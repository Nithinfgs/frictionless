# frictionless
QuickBox

A single-page web toolbox for everyday small tasks.

Features:


Clipboard history (saves and recalls copied text)
Text tools (case conversion, trim, dedupe, reverse, word count)
Quick to-do list
Unit converter (length, weight, time, data, temperature)
Password generator
Snippet vault for reusable text
Timestamp tools
QR code generator


Usage:
Open quickbox.html in a browser. All data is saved locally, no setup or login needed.
